Requirements
------------

This is a list of applications that need to be installed previously to
enjoy all the goodies of this configuration:

-  `Python 3.6.4`_
-  `Docker`_
-  `Docker Compose`_

.. code:: bash

    $ make setup
    $ make docker.build
